# MedFlicks - Medicine Reminder App

MedFlick is a user-friendly mobile application designed to help individuals manage their medications efficiently. It ensures that you never miss a dose by sending timely reminders and tracking your medicine intake. MedFlick is ideal for patients, caregivers, and anyone who needs assistance in maintaining a consistent medication schedule.


## Features

- **Personalized Medicine 
- **: Add multiple medications with custom schedules.
- **Push Notifications
- **: Receive reminders on time to take your medication.
- **Dosage Tracking
- **: Keep track of doses taken and missed.
- **Medication History
- **: View your past medication intake for better health monitoring.
- **User-Friendly Interface
- **: Simple and intuitive design for all age groups.
- **Customizable Alerts
- **: Choose your preferred notification sound and frequency.

## Installation

1. Clone the repository:
   git clone https://github.com/promeee100/MedFlick.git
2. Navigate to the project directory:

cd MedFlick

** Install dependencies:

flutter pub get

** Run the app:

flutter run


## Usage

1. Open MedFlick on your device.

2. Add a new medication by entering the name, dosage, and schedule.

3. Enable notifications to receive reminders.

4. Track your daily medication intake and view history.


## Technologies Used

            Flutter - Frontend framework

            Dart - Programming language

            Firebase - Database for storing medication data

            Push Notifications - For timely reminders
