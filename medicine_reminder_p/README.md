# MedFlicks Medicine Reminder App

Flutter source for an additional MedFlicks medicine reminder project included in this repository.

## Features

- Medication reminder concept for patients and caregivers.
- Medicine schedule management.
- Push-notification based reminder workflow.
- Medication intake tracking concept.
- Firebase-backed storage direction.

## Tech Stack

- Flutter
- Dart
- Firebase
- Push notifications

## Getting Started

```bash
flutter pub get
flutter run
```
